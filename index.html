HTML

  • Most basic building block of websites, handles the content of a website, as well as its structure and the meaning of each element on the page.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements are case-sensitive.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS box model
  • Card= class selector

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Git: version control system that allows web developers to collaborate on projects without overwriting each other's work.
  • GitHub: the largest provider of internet hosting for software and web development code, and provides the version control functionality of Git in a central repository.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The tool that takes a static HTML/CSS webpage and makes it interactive.
  • First item in any array has an index of 0, not 1.
  • For loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
  • You must call a function by name in order to execute it.
