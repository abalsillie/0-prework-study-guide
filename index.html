Alex's Prework Study Guide

HTML

  • Most basic building block of websites, handles the content of a website, as well as its structure and the meaning of each element on the page.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements are case-sensitive.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS box model
  • Card= class selector

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Git: version control system that allows web developers to collaborate on projects without overwriting each other's work.
  • GitHub: the largest provider of internet hosting for software and web development code, and provides the version control functionality of Git in a central repository.
